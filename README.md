# 🐍 Snake Game Project

## 📌 About the Project
This is a simple **Snake Game** built using Python and Pygame.  
The player controls the snake using arrow keys and eats food to increase score.

---

## 🎮 Features
- Smooth snake movement using keyboard arrows ⬅️➡️⬆️⬇️  
- Apple (food) eating system 🍎  
- Score tracking system 🏆  
- High score saved in file (`highscore.txt`)  
- Simple and interactive game window  

---

## 🛠️ Technologies Used
- Python 🐍  
- Pygame 🎮  
- File handling (for high score) 📁  
- Git & GitHub for version control 🌐
